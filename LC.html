Applying CSS

CSS can be applied to HTML in three ways: inline, internal, and external.

Inline CSS uses the style attribute in HTML tags, e.g., <p style="color: blue;">Text</p>. It’s quick but hard to maintain.

Internal CSS lives in a <style> tag in the <head> section. Example:

<style>
  p { color: blue; }
</style>

External CSS uses a separate .css file linked with <link>. Example:

<link rel="stylesheet" href="styles.css">

External CSS is best for larger projects due to reusability.

Selectors, Properties, and Values

CSS rules consist of selectors (which target HTML elements), properties (what to style), and values (how to style it). Example:

p {
  color: blue;
}

Here, p is the selector, color is the property, and blue is the value. Common selectors include element names (p, h1), classes (.class-name), and IDs (#id-name).

Colors

CSS supports colors in several formats: named colors (e.g., blue), hex (e.g., #3498db), RGB (e.g., rgb(52, 152, 219)), and RGBA for opacity (e.g., rgba(52, 152, 219, 0.5)).

h1 {
  color: #3498db;
}

Use colors to enhance visual appeal and ensure readability.

Text

CSS can style text with properties like font-size, font-family, font-weight, and text-align. Example:

p {
  font-size: 16px;
  font-family: Arial, sans-serif;
  text-align: center;
}

Experiment with text properties to improve readability and aesthetics.

Margins and Padding

Margins control the space outside an element, while padding controls the space inside an element. Example:

div {
  margin: 20px;
  padding: 15px;
}

Use margins and padding to create breathing room and improve layout.

Borders

Borders go around elements and are styled with border properties, like border-width, border-style, and border-color. Example:

div {
  border: 2px solid #3498db;
}

Borders help define boundaries and highlight elements.

Putting It All Together

Combine selectors, colors, text styles, margins, padding, and borders to style a webpage. Start with a simple HTML structure and apply CSS to make it visually appealing.

Recap: CSS is about styling HTML with selectors, properties, and values. Test your styles in a browser and keep experimenting!

Ready for more? Explore advanced CSS like flexbox, grid, or animations, and consider JavaScript for interactivity.
