Getting Started

HTML (HyperText Markup Language) is the foundation of web pages. It structures content using tags.

A basic HTML document looks like this:

<!DOCTYPE html>
<html>
<head>
  <title>My Page</title>
</head>
<body>
  <h1>Hello, World!</h1>
</body>
</html>

Page Titles

The <title> tag sets the page title, displayed in the browser tab.

<title>My Website</title>

Paragraphs

Use the <p> tag for paragraphs of text.

<p>This is a paragraph.</p>

Headings

Headings are defined with <h1> to <h6>, where <h1> is the largest.

<h1>Main Heading</h1>

Lists

Use <ul> for unordered lists and <ol> for ordered lists, with <li> for list items.

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

Links

Create hyperlinks with the <a> tag using the href attribute.

<a href="https://example.com">Visit Example</a>

Images

Add images with the <img> tag. Use src for the image path and alt for accessibility.

<img src="image.jpg" alt="Description">

Note: The <img> tag is self-closing.

Tables

Create tables with <table>, <tr> for rows, <th> for headers, and <td> for data.

<table>
  <tr>
    <th>Header</th>
    <th>Header</th>
  </tr>
  <tr>
    <td>Data</td>
    <td>Data</td>
  </tr>
</table>

Forms

Forms collect user input using <form>, with elements like <input> and <label>.

<form action="/submit">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">
  <input type="submit" value="Submit">
</form>

Putting It All Together

Combine these elements to create a structured webpage. Start with a basic layout and add content hierarchically.

Recap: HTML is about structuring content with tags. Test your page in a browser and keep experimenting!

Ready for more? Explore CSS for styling and JavaScript for interactivity.
